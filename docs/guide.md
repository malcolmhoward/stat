# S.T.A.T. - System Telemetry and Analytics Tracker

## Overview

S.T.A.T. is the telemetry backbone of the O.A.S.I.S. wearable computing platform. It monitors power, battery, CPU, memory, fan speed, and temperature, then broadcasts structured telemetry via MQTT for consumption by other components.
